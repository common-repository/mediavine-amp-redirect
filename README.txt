Redirect AMP traffic to the original page with a 302 Redirect
 
== Description ==
 
Decide you want to back out of AMP but you're still getting search traffic to your AMP pages? No problem! Simply install this alongside the standard [AMP plugin by Automattic](https://wordpress.org/plugins/amp/) and your pages will automatically be redirected to the non-amp page.

Built with Love by [Mediavine](https://mediavine.com/)
 
== Installation ==
  
Simply install & activate this plugin as you would with any wordpress plugin, and your AMP traffic will start redirecting
 
== Frequently Asked Questions ==
 
= Does this work for other AMP Plugins? =
 
While we officially support the AMP Plugin by Automattic, this should work with any AMP plugin
 
= Can I enable AMP pages later? =
 
Definitely! Simply deactivate the plugin and your AMP traffic will start going to AMP pages again.
